Gary AI Widget Status Check

Step 1: Check if Widget Container Exists

Visit your website at http://localhost:8080

Right-click on the page and select "View Page Source" or press Ctrl+U

Search for: gary-ai-widget-container

If you find this HTML:
<div id="gary-ai-widget-container" class="gary-ai-widget-bottom-right"></div>
→ The widget container is being added (chatbot is enabled)
If you DON'T find this HTML:
→ The chatbot is disabled (most likely cause)

Step 2: Check Browser Console

On your website, press F12 to open Developer Tools

Go to the Console tab and look for errors

Expected behavior if working:
• No JavaScript errors
• CSS file loads: assets/css/chat-widget.css
• JS file loads: assets/js/chat-widget.js
• Widget appears in bottom-right corner
